# Sponus is a CLI lyrics service


# Sponus (What it is)
  Type `sponus` in terminal and instantly get the lyrics of what you're listening on spotify. boom!


## Installation
1- clone the repository
  * `git clone https://github.com/mrtaalebi/sponus && cd sponus`

2- Run install.sh
  * `./install.sh`
